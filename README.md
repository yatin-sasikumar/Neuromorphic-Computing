# Low-Power Event-Driven Neuromorphic Design

## Overview

This project compares a traditional clock-driven FSM with an event-driven neuromorphic design for spatio-temporal event sequence detection.

The focus is on reducing power consumption and latency using event-driven computation.

---

## Key Results

*  Near-zero event-to-response delay
*  Dynamic power reduced from **0.39 mW → 0.01 mW (~97%)**
*  Comparable hardware resource usage

---

## Architecture

**FSM (Traditional):**

* Clock-driven
* Continuous switching
* Higher power, 1-cycle delay

**Neuromorphic:**

* Event-driven
* Sparse switching
* Low power, near-zero latency
